📌 Project Overview

This project was developed as a Capstone project/ Final Paper idealized, programmed and written by Lucas Pereira Alfaro, Sophie Sthephanie de Carvalho, Lorena de Castro and Thiago Falci. The project consists of a 2D plataformer mythology game, bringing an reimagening of Orpheus and Eurydice's myth. The game is a 2 player local multiplayer game that has speedrun elements such as a online scoreboard that marks the time that took the player to finish a level and puzzle game elements that require cooperation between player to pass different challenges in order to procede.

The game has the objective to be both appeal to gamers/mythology lovers but also to serve as a tool that can be use to teach younger children about greek mythology, logical thinking and history.
