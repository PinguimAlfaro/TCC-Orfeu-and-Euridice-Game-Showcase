📌 Project Overview

This project was developed as a Capstone project/ Final Paper programmed and written by Lucas Pereira Alfaro, Sophie Sthephanie de Carvalho, Lorena de Castro and Thiago Falci. The project consists of a 2D plataformer mythology game, inspired by the Orpheus and Eurydice's myth. The game is a 2 player local multiplayer game that has speedrun elements such as a online scoreboard that marks the time that took the player to finish a level and puzzle game elements that require cooperation between player to pass different challenges in order to procede.
